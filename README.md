# Vision Beats 🥁

**Vision Beats** is a virtual percussion studio built with **SwiftUI** and **AVFoundation**. By leveraging real-time camera tracking, the app transforms natural hand gestures into a responsive musical experience, allowing users to "air-drum" with studio type quality audio samples.

### Technical Highlights
* **SwiftUI & Materials:** A modern "glassmorphic" interface using `.ultraThinMaterial`.
* **Low-Latency Audio:** Optimized with **AVAudioPlayer** for instantaneous feedback.
* **Computer Vision:** High-speed motion detection via **AVFoundation**.
* **Accessibility:** Designed to replace physical drumsticks with intuitive spatial movements, making music creation silent, portable, and inclusive.

### How to use:
1. Open the `.swiftpm` project in **Xcode 16** or later.
2. Run the app on an ipad in landscape mode with a front-facing camera 
3. Position yourself in the frame and move your hands to strike the virtual drums!
